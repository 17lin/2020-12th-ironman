KNN 的全名 K Nearest Neighbor 是屬於機器學習中的Supervised learning其中一種算法
 ，顧名思義就是 k 個最接近你的`鄰居`。分類的標準是由鄰居「多數表決」決定的。在 sklearn 中 KNN 可以用作分類或迴歸的模型。
