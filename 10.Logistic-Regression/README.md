羅輯回歸(Logistic regression)是由線性回歸變化而來的。線性回歸是用來預測一個連續的值，而邏輯回歸是用來處理分類問題。邏輯回歸利用sigmoid function 轉換成0~1的值，表示可能為這個類別的機率值。
