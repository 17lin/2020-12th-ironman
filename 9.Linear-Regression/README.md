## 線性回歸（Linear regression）
線性回歸是統計上在找多個自變數和依變數之間的關係所建出來的模型。只有一個自變數(x)和一個依變數(y)的情形稱為簡單線性回歸大於一個自變數(x1,x2,...)的情形稱為多元回歸。
