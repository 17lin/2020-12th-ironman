支持向量機（SVM）是專門處理分類的問題，還有另一個名詞稱為支持向量迴歸（Support Vector Regression, SVR）專門處理迴歸問題。SVR 是 SVM 的延伸，而支持向量迴歸只要 `f(x)` 與 y 偏離程度不要太大，既可以認為預測正確。

> 只有在虛線以外的誤差才會被計算
