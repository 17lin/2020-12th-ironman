## 集成學習
Ensemble learning 又稱集成學習，指的是以一個系統化的方式將好幾個監督式學習的模型結合在一起，目的是希望結合眾多的模型產生一個更強大的模型。在許多科學競賽中Ensemble learning在實務上是非常有效的提升預測準確率。依照Ensemble的處理方式的不同，我們可以將它分為三類。第一類為Bagging，第二類為Boosting，第三類為Stacking。

- Bagging:
    - Random forest
- Boosting:
    - AdaBoost
    - Gradient Boosting 
    - XGBoost
- Stacking
